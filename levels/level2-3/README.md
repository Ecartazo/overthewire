---

## 🧠 Problema clave

El archivo tiene dos complicaciones:
- Contiene espacios
- Empieza con `--`, lo que hace que Linux lo interprete como una opción

---

## 🧠 Conceptos
- Uso de comillas `" "` para manejar espacios
- Uso de `--` para finalizar opciones en comandos
